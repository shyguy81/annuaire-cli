---
name: deploy-local
description: Build annuaire-cli in release mode and install the binary to ~/.local/bin for local testing.
---

Compile annuaire-cli en release et installe le binaire dans `~/.local/bin` (ajoute ce dossier au PATH si absent).

Exécuter: `./scripts/deploy_locally.sh`

Après exécution, vérifier avec `annuaire-cli --help`.
